# EMIASback


На чём написанна:
- [Express](https://github.com/expressjs/express)
- [Taki DataBase](https://github.com/nik19ta/tdb)



Запуск:

```sh
npm run start # node app.js
```

Запуск для разработки:

```sh
npm run dev # nodemon app.js
```
